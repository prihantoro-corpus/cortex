# Wordlist Directory

This directory contains frequency lists organized by language, which can be used for keyword analysis.

## Structure

Each language has its own subdirectory:
- `indonesian/` - Indonesian frequency lists
- `english/` - English frequency lists
- `arabic/` - Arabic frequency lists
- `chinese/` - Chinese frequency lists
- `hindi/` - Hindi frequency lists
- `japanese/` - Japanese frequency lists
- `javanese/` - Javanese frequency lists
- `korean/` - Korean frequency lists

## Supported Formats

Frequency lists should be in one of the following formats:
- `.txt` - Plain text format (word frequency pairs)
- `.csv` - CSV format (word, frequency columns)
- `.xlsx` - Excel format

## Usage

These frequency lists are used as reference corpora for keyword analysis, allowing users to compare their corpus against established frequency distributions for each language.
